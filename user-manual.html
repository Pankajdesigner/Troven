Create a Workspace

Workspaces are folders in which you can create assessments.

Follow below steps to create a Workspace:

  • Navigate to Workspace tab
  • Click on New Workspace
  • Add Workspace name, Description, Thumbnail and save it.
  • Once a Workspace is created, you will see the following information on the Workspace card.
    • Created by: Name of the user who created the workspace.
    • Last Visited: Duration of the last visited
    • Role based assessments: Number of role based assessments created in the workspace
    • Skill based assessments: Number of skill based assessments created in the workspace
    • Limit of the Workspace: Each workspace has a limit of creating 10 assessments each. It will show the number of assessments created out of 10.

Create a Student batch

A folder in which you can add candidates either one by one or in bulk whom you want to invite for an assessment

Follow below steps to create a Student batch

  • Navigate to Student Management tab
  • Click on New Batch
  • Add Student batch name, Description, Thumbnail and save it.
  • Once a Student batch is created, you will see the following information on the Student batch card
    • Created by: Name of the user who created the Student batch.
    • Number of students: Number of students added in the batch.

Add students
You can add candidates either one by one or import in bulk

Follow below steps to add students One by one.

  • Open a Student batch
  • Click on Add student
  • Add student details Name, email, phone number and click on Add

Follow below steps to import students in bulk.

  • Open a Student batch
  • Click on Import students
  • Download the sample CSV file.
  • In the CSV file, add the list of students with their info - Name, Email and Phone number and save it.
  • After you updated the candidate’s details in the file, Follow steps 1 and 2 again and then upload the CSV file that you updated and click on Add

Create an Assessment

You can create an assessment using your own questions or import from Troven’s library

Create your Questions library
You can bulk upload questions in the Question Bank and create your own Questions library.

Follow below steps to Bulk upload questions in the Question Bank.

  • Navigate to Questionbank.
  • Click on New Folder you will get a pop up, Enter the folder name and click on Create folder.
  • Add Student batch name, Description, Thumbnail and save it.
  • In the folder you will see Import questions and Generate using AI.
  • To import questions in Bulk, click on Import questions,You will see a pop up. Download sample csv file from the pop up.
  • After updating the CSV file, go back to the pop up, Select Skill based if your file consists of questions for only one skill. If your file has questions for more than one skill then choose Role based. To upload the csv file, Click on Upload file, browse to the file in your computer and then click on Upload.
  • Once the csv file is uploaded, all your questions will be loaded in the folder. You can edit or delete your questions from here by clicking on the specific option for each question.
  • Click Back to navigate to the Question Bank.

Assessment Setting
Follow below steps to add students One by one.

  • Open a workspace.
  • Click on New Assessment

  • In settings, Add Assessment details:
    • Assessment Name: Title of your assessment
    • Choose Invite template: In the email template section you can customize the invite template.
    • Add duration of the assessment: Total time for the assessment
    • Difficulty level: Easy, Medium, Hard
    • Start date: Start date and time of the assessment
    • End date: End date and time of the assessment
    • Total marks: Total marks allotted for the assessment
    • Pass marks: Marks to be obtained to qualify the assessment
    • Negative marking (if any): Penalty on wrong answers
  • In settings, Select Permission preference
    • Disable clipboard:
    • Submit on tab switch:
    • Enable microphone:
    • Start with Full screen:
    • Enable webcam:
    • MFA verification:
  • In settings, Select Questions order
    • You can click and drag to move the sections above or below based on your preference.
  • In settings, Add Instructions:
    • These are the instructions for taking the assessment.It will be displayed before a candidate starts the assessment.
  • In settings, Select Student Report Visibility
    • You choose to Show or hide the assessment report to the student using the check box.
    • You can choose to Show or hide the assessment questions in the Student report to the student using the checkbox.
  • Save changes before you move on to the next tab.

Creating custom coding challenges
Add your question one by one:

  • Click on Coding challenges tab in the Assessment setting window
  • On the ride side you will see a Create challenge option. Using this option you can create your own custom coding challenge. Click on Create challenge.
  • A new page opens in which you can start with adding the difficulty level of the coding challenge, marks you want to allot for it, Enter your question, Add number of parameters in your coding challenges, if there are two then Varx and Vary.
  • Examples: Add examples for your coding challenges, If the challenge is to add two numbers then add two numbers separated by a comma in the Sample input section and its output based on the challenge in the Sample output section.
  • Test cases: These test cases will run in the background and a candidate will not have access to see them. Add input and output similar to the examples as mentioned above. Question description: Add more details in this section(Optional)
  • Total marks: Total marks allotted for the assessment.
  • Available marks: Total marks available which can be used for more questions. Eg. If total marks are 10, you have added 2 questions with 2 marks each there you will be left with 6 marks. In this case 6 marks becomes the available marks which you can use for more questions.
  • Starting code: In this section, first select the language for your coding challenge. Add basic code structure for the candidate. An incomplete code which will be completed by the candidate in the assessment. However you can put the complete code first, click on Run to test it.
  • Output: In the output section you can see the results of the coding challenge.
  • After adding all the details, click on create or cancel if you don't want to add the coding challenge.

Add questions using Your library

  • Click on Coding challenges tab in the Assessment setting window
  • On the left you will see Your library, Click on it you will see all the coding challenges from your Library that you uploaded in Your Question Bank.
  • To filter the coding challenges, you can select a specific question library from your Question Bank using the dropdown menu.
  • Filter the difficulty level.
  • Further filter by selecting the Skills
  • After you select all your preferred filters, you will see Coding challenges from your library in the center of the window.
  • Use search for further shortlisting.
  • To add a coding challenge into your assessment, click on the + symbol. The selected coding challenge will appear at the ride side of the screen.
  • In the marks section, add marks for the coding challenge individually.
  • Click on Save after you are done with adding the coding challenges for the Assessment.

  • Enter your question

  • To mark the correct option click on the bubble. It will turn green
  • To mark the correct option click on the bubble. It will turn green

Clone Assessments from Troven Library You can clone assessments directly from the Troven’s library and also customize it further.

Follow below steps to Clone assessments from Troven Library.

  • In a workspace you will have an option to Create Assessment. Click on Create Assessment
  • Choose Assessment type - Skill Based or Role based. Learn more
  • If creating a Skill based assessment, select one skill
  • Click on Clone from Troven
  • In a new window you will see all the assessments from the Troven’s library based on your selection
  • Click on preview to see the assessments questions and details
  • Click on Clone to clone the assessment from Troven’s library
  • An assessment setting page will open in which you can make further changes.

Custom email templates

Customize communication emails for your candidates

Follow below steps to add Customized email templates

  • Click on “Email templates” from the left navigation menu.
  • There email templates that you can customize. They are Invitation Template, Welcome Template, Reminder Template, Result Template, Completion Template, Interview Template, Verification email - OTP.
  • To update any template, click on the template name on the left.
  • On the right window you will see the Create Template option. Click on it
  • A window will pop up in which you will get options to add
    • Name: Name of the template
    • Subject: Add a short line for candidates to recognize the email context
    • Body: Add your communication content in it
  • Click on Save to save your template
  • Repeat the above step to edit other templates as well

User Management

A tenant has an option to add multiple users from their organization who can take various responsibilities of updating, analyzing or monitoring the assessment process.

Follow below steps to Add a user to your Tenant profile

  • Click on “User Management” from the left navigation menu
  • Click on “Add user”, a pop will come up with following options - Name, Mobile Number, Email, Password and Role
  • Role - There are 3 roles - Super Admin, Admin and Creator
  • Super Admin: You can add multiple Super Admins based on your subscription plan. A Super Admin will have access to all the features including updating Subscription, Deleting any User, Candidates etc.
  • Admin: Admin wll access to most of features excluding, Updating Subscription, Adding or Deleting any User.
  • Creator: A creator will be limited to creating the assessments and any content required on the platform for Assessment
  • After adding users, you will see the user list as below
  • Only a Super Admin has the option to Enable/Disable, Edit or Delete a user
